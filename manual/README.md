# manual 폴더 사용법

이 폴더는 수동으로 클립보드 데이터를 파일로 복사하거나 붙여넣는 스크립트를 제공합니다.

## 디렉터리 구조

- `sender/` : 파일을 base64로 인코딩하여 복사하는 스크립트가 있습니다.
- `recv/`   : base64로 인코딩된 파일을 붙여넣는 스크립트가 있습니다.

## sender/copy_b64_files.py
- 지정한 파일을 base64로 인코딩하여 클립보드에 복사합니다.
- 사용 예시:
  ```bash
  python copy_b64_files.py <파일경로>
  ```
- 여러 파일을 한 번에 복사할 수 있습니다.

## recv/paste_b64_files.py
- 클립보드에 복사된 base64 데이터를 파일로 저장합니다.
- 사용 예시:
  ```bash
  python paste_b64_files.py <저장할_디렉터리>
  ```
- 여러 파일을 한 번에 붙여넣을 수 있습니다.

## 참고 사항
- 이 스크립트들은 base64 인코딩/디코딩을 사용하여 파일을 안전하게 복사/붙여넣기 할 수 있도록 도와줍니다.
- 윈도우 환경에서 동작하도록 설계되었습니다.
- 파이썬 3이 필요합니다.
